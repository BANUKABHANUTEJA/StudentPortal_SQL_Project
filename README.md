# StudentPortal_SQL_Project

Student Portal Database Ops + Reports

Tools Used:
- SQL Server
- SSMS

How to Run:
1. Run create_tables.sql
2. Run 01_dml_operations.sql
3. Run 02_reports.sql
4. Export results to output/report_results.csv

Features:
- Insert Students
- Update Marks
- Delete Duplicate Records
- Transactions with Rollback
- Reports using COUNT, AVG, SUM, GROUP BY, HAVING
